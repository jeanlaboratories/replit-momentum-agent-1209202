# **App Name**: AdVantage

## Core Features:

- Brand Profile Generator: Scrape the website text and metadata to understand the brand's voice and offerings and creates a brand profile.
- Campaign Timeline Editor: A visual, drag-and-drop timeline editor for creating multi-day marketing campaigns.
- Content Block Creation: Add 'content blocks' (Social Media Post, Email Newsletter, Blog Post Idea) to each day, specifying the message, tone, and desired visuals.
- AI Content Generation Tool: Generates campaign content using a LLM 'tool', including ad copy, images, and short video clip prompts, based on the brand profile and user inputs.
- AI Image Generation: Leverages AI to generate images for the campaigns based on the brand profile and campaign description.
- Campaign Preview and Editing: Displays a preview of the generated campaign, allowing users to make minor edits to the text.
- Workspace/Network Management: Allows users to create and manage company networks/workspaces.

## Style Guidelines:

- Primary color: Deep violet (#673AB7), evoking creativity and sophistication.
- Background color: Light grey (#F0F2F5), to ensure readability and reduce eye strain.
- Accent color: Soft lavender (#BB86FC), for highlights and interactive elements.
- Headline font: 'Space Grotesk' sans-serif. Body Font: 'Inter' sans-serif
- Use minimalist icons with a thin stroke to represent content types and actions.
- Employ a clean, card-based layout to organize campaigns and content blocks.
- Use subtle transitions and animations to enhance the user experience, such as content blocks smoothly expanding when selected.
