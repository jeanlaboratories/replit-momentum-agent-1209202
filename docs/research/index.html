The Momentum Metaphor

In physics, momentum represents an object's resistance to stopping once in motion. We adopt this metaphor for our agent architecture.

p = m Ã— v
m = Context
Team knowledge, brand guidelines, user preferences, conversation history
v = Model
Execution speed and quality of foundation models (Gemini, Imagen, Veo)
p = Momentum
Unstoppable, contextually-rich task completion

Key Innovations

ğŸ”„

Hierarchical Context Injection

Five-layer context system (Brand, User, Settings, Media, Team) that propagates through all tool invocations via thread-safe global state, ensuring context is never lost between tool transitions.

ğŸ§ 

Team Intelligence Pipeline

Automated extraction of structured insights from heterogeneous artifacts (PDFs, websites, social media) into a unified Brand Soul representation for consistent generation.

ğŸ’¾

Dual-Layer Memory

Vertex AI Memory Bank for semantic search with Firestore fallback, enabling persistent personalization that survives session boundaries with graceful degradation.

ğŸ¨

Character Consistency

Integration of reference image composition (up to 14 images) enabling visually coherent multi-asset campaign generation with consistent characters across all outputs.

ğŸ”

Multi-Agent Search

Dedicated search sub-agent with built-in google_search solves Gemini's tool-mixing limitation while preserving custom tool flexibility in the main agent.

ğŸ“Š

Context Flow Evaluation

Novel metrics for measuring semantic preservation across tool transitions, including Context Perplexity and Cross-Modal Coherence scores.

Built with Google Agent Development Kit

MOMENTUM is built on the Google Agent Development Kit (ADK), a flexible and modular framework for developing and deploying AI agents. ADK provides:

  • Model-agnostic agent architecture optimized for Gemini
  • Multi-agent patterns with AgentTool delegation
  • Built-in tools including google_search
  • Vertex AI Memory Bank for persistent memory
  • Streaming response support
  • Session management and state persistence
