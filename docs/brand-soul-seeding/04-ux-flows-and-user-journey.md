# Brand Soul Seeding - UX Flows & User Journey

## User Personas

### Primary Users

**1. Marketing Manager (Maya)**
- **Role**: Manager at tech startup
- **Goals**: Establish consistent brand voice, create campaigns efficiently
- **Pain Points**: Brand messaging inconsistency, time-consuming content creation
- **Tech Savvy**: High
- **Time Available**: Limited, wants quick setup

**2. Small Business Owner (Sam)**
- **Role**: Owner of local coffee shop
- **Goals**: Build brand online, create social media content
- **Pain Points**: No marketing experience, doesn't know where to start
- **Tech Savvy**: Medium
- **Time Available**: Very limited, prefers guided workflows

**3. Brand Strategist (Bethany)**
- **Role**: Contributor at marketing agency
- **Goals**: Deep brand research, comprehensive brand books
- **Pain Points**: Manual research is time-consuming, scattered sources
- **Tech Savvy**: High
- **Time Available**: Has time for thorough setup

## Mobile Experience

### Mobile-First Considerations

**Simplified Quick Start (Mobile)**:
```
1. Welcome (swipe carousel)
2. Website URL (large input, keyboard optimization)
3. Skip media kit (too complex for mobile upload)
4. Manual notes (voice input option)
5. Processing (background, notification when done)
6. Review (simplified cards, thumb-scroll)
7. Approve (single large button)
```

**Mobile Dashboard**:
- Card-based layout
- Swipe gestures
- Bottom sheet modals
- Pull-to-refresh
- Push notifications for completion
